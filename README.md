### Controls
##### BASIC MOVEMENT
 - WASD:         movement
 - SPACE:        jump
 - LEFT SHIFT:   sprint
 - LEFT CTRL:    crouch or slide (while running)

##### SHOOTPLAY
 - R:            reload
 - LEFT CLICK:   shoot
 - RIGHT CLICK:  zoom
 - Q:            change weapon / ammo type (gear wheel)
