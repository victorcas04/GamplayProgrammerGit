### Controls
##### BASIC MOVEMENT
 - WASD:         movement
 - SPACE:        jump
 - LEFT SHIFT:   sprint
 - LEFT CTRL:    crouch or slide (while running)

##### SHOOTPLAY
 - R:            reload (WIP)
 - LEFT CLICK:   shoot
 - RIGHT CLICK:  zoom

##### OTHER
 - E:            interact (WIP)
 - F:            use ability (WIP)
 - Q:            open gear wheel or change weapon (WIP)
